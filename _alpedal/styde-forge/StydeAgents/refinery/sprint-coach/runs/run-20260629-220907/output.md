Coaching tip for today:
Inspect your cycle time distribution before sprint planning. If your 85th percentile cycle time exceeds sprint length, slice user stories smaller. A story that takes the whole sprint is a project, not a story. Target 2-3 stories per developer per sprint for predictable delivery.
