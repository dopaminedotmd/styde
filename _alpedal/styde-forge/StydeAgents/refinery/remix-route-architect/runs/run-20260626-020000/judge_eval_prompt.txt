## 1. Architecture Overview

This blueprint extends the Cycle 1 Remix v2+ architecture with **five advanced patterns** that push Remix beyond its conventional SSR model into SPA-like instant navigation, real-time streaming data, and bi-directional WebSocket communication.

```typescript
// app/lib/api.ts
const API_BASE = import.meta.env.VITE_API_URL ?? "/api";

class ApiError extends Error {
  constructor(
    public status: number,
    message: string,
    public data?: unknown
  ) {
    super(message);
    this.name = "ApiError";
  }
}

export async function apiFetch<T = unknown>(
  path: string,
  init?: RequestInit
): Promise<T> {
  const res = await fetch(`${API_BASE}${path}`, {
    headers: {
      "Content-Type": "application/json",
      ...init?.headers,
    },
    ...init,
  });

  if (!res.ok) {
    const body = await res.json().catch(() => ({}));
    throw new ApiError(res.status, body.message ?? res.statusText, body);
  }

  return res.json();
}
```

## 3. clientLoader — Instant Optimistic Transitions

`clientLoader` runs **only in the browser** and can return cached data before the server loader resolves. This eliminates network waterfalls for back/forward navigation and enables instant page transitions.

### 3.1 The Pattern

When a user navigates to a route, Remix runs `clientLoader` first (if JS is available). If it returns data, the page renders immediately. Then the server `loader` runs and hydrates with fresh data if needed.

```
Navigation happens
     │
     ├── clientLoader ──→ cache hit? ──→ render instantly
     │                                         │
     │    cache miss ──→ fall through ─────────┤
     │                                         │
     └── server loader ──→ DB query ──→ hydrate/replace data
```

### 3.2 clientLoader with LRU Cache

```typescript
// app/lib/cache.client.ts — Client-side cache (browser only)
interface CacheEntry<T> {
  data: T;
  timestamp: number;
  ttl: number;
}

class ClientCache {
  private store = new Map<string, CacheEntry<unknown>>();
  private maxSize: number;

  constructor(maxSize = 100) {
    this.maxSize = maxSize;
  }

  get<T>(key: string): T | undefined {
    const entry = this.store.get(key);
    if (!entry) return undefined;

    // TTL check
    if (Date.now() - entry.timestamp > entry.ttl) {
      this.store.delete(key);
      return undefined;
    }

    // LRU: move to end
    this.store.delete(key);
    this.store.set(key, entry);
    return entry.data as T;
  }

  set<T>(key: string, data: T, ttlMs = 30_000): void {
    // Evict oldest if at capacity
    if (this.store.size >= this.maxSize) {
      const oldest = this.store.keys().next().value;
      if (oldest) this.store.delete(oldest);
    }

    this.store.set(key, {
      data,
      timestamp: Date.now(),
      ttl: ttlMs,
    });
  }

  invalidate(pattern: RegExp): void {
    for (const key of this.store.keys()) {
      if (pattern.test(key)) this.store.delete(key);
    }
  }

  clear(): void {
    this.store.clear();
  }
}

export const clientCache = new ClientCache(200);
```

```tsx
export async function clientLoader({
  params,
  serverLoader,
}: ClientLoaderFunctionArgs): Promise<LoaderData> {
  const cacheKey = `project:${params.id}`;

  // 1. Check client cache first (instant if hit)
  const cached = clientCache.get<Project>(cacheKey);
  if (cached) {
    // Still call serverLoader in background for fresh data

      clientCache.set(cacheKey, data.project, 60_000); // 1 min TTL
      // React will re-render when the promise resolves if data differs

  }

  // 2. Cache miss: wait for server
  const data = await serverLoader();
  clientCache.set(cacheKey, data.project, 60_000);
  return data;
}
```

## 4. Server-Sent Events — Live Data Streams

SSE provides a unidirectional stream from server to client over a single HTTP connection. Perfect for activity feeds, notifications, and real-time dashboards without the complexity of WebSockets.

### 4.1 SSE Server Route (Custom Hono Server)

```typescript
// app/server/sse.ts
import type { Context } from "hono";
import { EventEmitter } from "node:events";

// ── Global event bus ───────────────────────────────────────────
export const sseEvents = new EventEmitter();
sseEvents.setMaxListeners(500);

// ── SSE message types ──────────────────────────────────────────
export type SSEEvent =
  | { type: "project:created"; project: { id: number; name: string } }
  | { type: "project:updated"; project: { id: number; name: string } }
  | { type: "project:deleted"; id: number }
  | { type: "member:joined"; projectId: number; user: { id: number; name: string } }
  | { type: "member:left"; projectId: number; userId: number }
  | { type: "comment:new"; projectId: number; comment: { id: number; body: string; author: string } }
  | { type: "heartbeat"; timestamp: number };

// ── SSE connection handler ─────────────────────────────────────
export async function handleSSE(c: Context) {
  const projectId = c.req.query("projectId");
  const token = c.req.query("token");

  // Authenticate (simplified — use real JWT/session validation)
  if (!token || !validateToken(token)) {
    c.status(401);
    return c.json({ error: "Unauthorized" });
  }

  // Set SSE headers
  c.header("Content-Type", "text/event-stream");
  c.header("Cache-Control", "no-cache, no-transform");
  c.header("Connection", "keep-alive");
  c.header("X-Accel-Buffering", "no"); // Disable nginx buffering

  // Create stream
  const stream = new ReadableStream({
    start(controller) {
      const encoder = new TextEncoder();

      // Helper to send an SSE message
      const sendEvent = (event: SSEEvent) => {
        // Filter by projectId if specified
        if (
          projectId &&
          "projectId" in event &&
          event.projectId !== Number(projectId)
        ) {
          return;
        }
        if ("project" in event && event.project.id !== Number(projectId) && projectId) {
          return;
        }

        const data = JSON.stringify(event);
        const message = `event: ${event.type}\ndata: ${data}\n\n`;
        controller.enqueue(encoder.encode(message));
      };

      // Send initial connection confirmation
      const initMessage = `event: connected\ndata: ${JSON.stringify({ timestamp: Date.now() })}\n\n`;
      controller.enqueue(encoder.encode(initMessage));

      // Subscribe to events
      const handler = (event: SSEEvent) => sendEvent(event);
      sseEvents.on("event", handler);

      // Heartbeat every 30 seconds to keep connection alive
      const heartbeat = setInterval(() => {
        sendEvent({ type: "heartbeat", timestamp: Date.now() });
      }, 30_000);

      // Cleanup on disconnect
      c.req.raw.signal.addEventListener("abort", () => {
        clearInterval(heartbeat);
        sseEvents.off("event", handler);
        controller.close();
      });
    },
  });

  return c.body(stream);
}

// ── Helper: emit events from anywhere in the app ──────────────
export function emitSSE(event: SSEEvent): void {
  sseEvents.emit("event", event);
}

// ── Token validation placeholder ───────────────────────────────
function validateToken(token: string): boolean {
  // In production: verify JWT or session token
  return token.length > 10;
}
```

### 4.2 SSE Client Hook

```typescript
// app/hooks/useSSE.ts
import { useEffect, useRef, useState } from "react";

interface UseSSEOptions {
  projectId?: number;
  token?: string;
  enabled?: boolean;
}

interface UseSSEReturn {
  events: SSEEvent[];
  connected: boolean;
  lastEvent: SSEEvent | null;
  error: string | null;
}

export function useSSE(options: UseSSEOptions = {}): UseSSEReturn {
  const { projectId, token, enabled = true } = options;
  const [events, setEvents] = useState<SSEEvent[]>([]);
  const [connected, setConnected] = useState(false);
  const [lastEvent, setLastEvent] = useState<SSEEvent | null>(null);
  const [error, setError] = useState<string | null>(null);
  const eventSourceRef = useRef<EventSource | null>(null);

  useEffect(() => {
    if (!enabled || !token) return;

    const params = new URLSearchParams();
    if (projectId) params.set("projectId", String(projectId));
    params.set("token", token);

    const url = `/sse?${params.toString()}`;
    const es = new EventSource(url);
    eventSourceRef.current = es;

    es.addEventListener("connected", () => {
      setConnected(true);
      setError(null);
    });

    // Listen for all known event types
    const knownTypes = [
      "project:created",
      "project:updated",
      "project:deleted",
      "member:joined",
      "member:left",
      "comment:new",
      "heartbeat",
    ];

    for (const type of knownTypes) {
      es.addEventListener(type, (event) => {
        const data = JSON.parse(event.data) as SSEEvent;
        setEvents((prev) => [...prev.slice(-99), data]); // Keep last 100
        setLastEvent(data);
      });
    }

    es.addEventListener("error", (ev) => {
      // EventSource will auto-reconnect; just track state
      setConnected(false);
      if (es.readyState === EventSource.CLOSED) {
        setError("SSE connection closed");
      }
    });

    return () => {
      es.close();
      eventSourceRef.current = null;
      setConnected(false);
    };
  }, [enabled, projectId, token]);

  return { events, connected, lastEvent, error };
}
```

```tsx
interface ActivityItem {
  type: string;
  message: string;
  timestamp: number;

function mergeActivity(
  initial: ActivityItem[],
  sseEvents: SSEEvent[]
): ActivityItem[] {
  const sseItems = sseEvents.map((e) => ({
    type: e.type,
    message: formatSSEEvent(e),
    timestamp: (e as any).timestamp ?? Date.now(),
  }));
  return [...sseItems.reverse(), ...initial].slice(0, 50);
}

function formatSSEEvent(event: SSEEvent): string {
  switch (event.type) {
    case "project:created":
      return `Project "${event.project.name}" was created`;
    case "project:updated":
      return `Project "${event.project.name}" was updated`;
    case "project:deleted":
      return `Project #${event.id} was deleted`;
    case "member:joined":
      return `${event.user.name} joined the project`;
    case "member:left":
      return `A member left the project`;
    case "comment:new":
      return `${event.comment.author} commented: "${event.comment.body.slice(0, 80)}"`;
    default:
      return "Unknown event";
  }
}

function formatTimeAgo(ts: number): string {
  const sec = Math.floor((Date.now() - ts) / 1000);
  if (sec < 60) return "just now";
  if (sec < 3600) return `${Math.floor(sec / 60)}m ago`;
  if (sec < 86400) return `${Math.floor(sec / 3600)}h ago`;
  return `${Math.floor(sec / 86400)}d ago`;
}

function getAuthToken(): string {
  // In SPA mode, read from localStorage or cookie
  return localStorage.getItem("auth-token") ?? "";
}
```

## 5. defer / Suspense / Await — Streaming SSR

`defer()` lets you stream slow data to the client while immediately sending the page shell. The browser renders the page instantly, then fills in deferred data as it arrives.

### 5.1 The Streaming Pattern

```
Request: GET /app/projects/42
     │
     ├── loader returns defer({ project: projectPromise, … })
     │         │
     │         ├── critical data resolved instantly ──→ sent in initial HTML
     │         │    (page shell: header, sidebar, tabs)
     │         │
     │         └── slow data resolves later ──→ streamed as JSON chunk
     │              (activity chart, large dataset, external API)
     │
     └── Browser renders shell immediately with <Suspense> fallbacks
          Then replaces fallbacks as deferred data arrives
```

### 5.2 Deferred Loader with Typed Defer

```typescript
// app/lib/defer.ts — Typed wrapper for Remix defer
import { defer as remixDefer } from "@remix-run/node";

/**
 * Typed defer() — preserves type information for useLoaderData<typeof loader>()
 */
export function defer<T extends Record<string, unknown>>(
  data: T
): TypedDeferredResponse<T> {
  return remixDefer(data) as TypedDeferredResponse<T>;
}

// Type utility (simplified — in production use full Remix types)
type TypedDeferredResponse<T> = ReturnType<typeof remixDefer> & { __type: T };
```

### 5.4 defer Best Practices

| Guideline | Why |
|---|---|
| **Defer slow, non-critical data** | Sends the page shell in ~100ms, streams the rest |
| **Catch promise rejections** | `.catch(() => fallback)` prevents killing the entire response |
| **Use `errorElement` on `Await`** | Graceful error handling per deferred stream |
| **Skeleton fallbacks** | Match the shape of the real content for minimal layout shift |
| **Multiple `Suspense` boundaries** | Each deferred promise gets its own boundary; load independently |
| **Don't defer authentication** | Auth checks must resolve before rendering; use regular loaders |
| **Combine with `clientLoader`** | Cache deferred results on the client for instant re-visits |

```typescript
// ── WebSocket server ───────────────────────────────────────────
interface WSClient {
  ws: WebSocket;
  userId?: number;
  projectId?: number;
  subscribed: boolean;
}

const wsClients = new Map<WebSocket, WSClient>();

function setupWebSocket(server: ReturnType<typeof createServer>) {
  const wss = new WebSocketServer({ server, path: "/ws" });

  wss.on("connection", (ws, req) => {
    const url = new URL(req.url ?? "/", "http://localhost");
    const token = url.searchParams.get("token");
    const projectId = url.searchParams.get("projectId");

    // Authenticate
    const user = authenticateWS(token);
    if (!user) {
      ws.close(4001, "Unauthorized");
      return;
    }

    const client: WSClient = {
      ws,
      userId: user.id,
      projectId: projectId ? Number(projectId) : undefined,
      subscribed: false,
    };
    wsClients.set(ws, client);

    console.log(`🔌 WS connected: user=${user.id} project=${projectId}`);

    // Send connection confirmation
    ws.send(
      JSON.stringify({
        type: "connected",
        payload: { userId: user.id, timestamp: Date.now() },
      })
    );

    // Handle incoming messages
    ws.on("message", (raw) => {
      try {
        const message = JSON.parse(raw.toString());
        handleWSMessage(client, message);
      } catch (err) {
        ws.send(
          JSON.stringify({
            type: "error",
            payload: { message: "Invalid message format" },
          })
        );
      }
    });

    // Handle disconnect
    ws.on("close", (code, reason) => {
      console.log(
        `🔌 WS disconnected: user=${user.id} code=${code} reason=${reason}`
      );
      wsClients.delete(ws);

      // Notify project members
      if (client.projectId) {
        broadcastToProject(client.projectId, {
          type: "user:offline",
          payload: { userId: user.id },
        });
      }
    });

    // Handle errors
    ws.on("error", (err) => {
      console.error(`WS error for user=${user.id}:`, err.message);
      wsClients.delete(ws);
    });
  });

  console.log("🔌 WebSocket server ready on /ws");
  return wss;
}

// ── WebSocket message handler ──────────────────────────────────
function handleWSMessage(
  client: WSClient,
  message: WSIncomingMessage
): void {
  switch (message.type) {
    case "ping": {
      client.ws.send(JSON.stringify({ type: "pong", payload: { timestamp: Date.now() } }));
      break;
    }

    case "subscribe:project": {
      const projectId = message.payload?.projectId;
      if (!projectId) {
        client.ws.send(
          JSON.stringify({
            type: "error",
            payload: { message: "Missing projectId" },
          })
        );
        return;
      }
      client.projectId = projectId;
      client.subscribed = true;

      // Notify project members
      broadcastToProject(projectId, {
        type: "user:online",
        payload: { userId: client.userId, projectId },
      });

      client.ws.send(
        JSON.stringify({
          type: "subscribed",
          payload: { projectId },
        })
      );
      break;
    }

    case "cursor:move": {
      if (!client.projectId) return;
      broadcastToProject(client.projectId, {
        type: "cursor:move",
        payload: {
          userId: client.userId,
          x: message.payload?.x ?? 0,
          y: message.payload?.y ?? 0,
        },
      });
      break;
    }

    case "comment:add": {
      if (!client.projectId) return;
      const comment = {
        id: Date.now(),
        body: message.payload?.body ?? "",
        authorId: client.userId,
        authorName: message.payload?.authorName ?? "Unknown",
        timestamp: Date.now(),
      };

      // Emit via SSE for dashboard viewers
      emitSSE({
        type: "comment:new" as any,
        projectId: client.projectId,
        comment: {
          id: comment.id,
          body: comment.body,
          author: comment.authorName,
        },
      });

      // Broadcast to WebSocket clients in the same project
      broadcastToProject(client.projectId, {
        type: "comment:add",
        payload: comment,
      });
      break;
    }

    case "project:update": {
      if (!client.projectId) return;
      broadcastToProject(client.projectId, {
        type: "project:update",
        payload: message.payload,
      });
      break;
    }

    default: {
      client.ws.send(
        JSON.stringify({
          type: "error",
          payload: { message: `Unknown message type: ${message.type}` },
        })
      );
    }
  }
}

// ── Broadcast helpers ──────────────────────────────────────────
function broadcastToProject(
  projectId: number,
  message: WSOutgoingMessage
): void {
  for (const [, client] of wsClients) {
    if (
      client.projectId === projectId &&
      client.ws.readyState === WebSocket.OPEN
    ) {
      client.ws.send(JSON.stringify(message));
    }
  }
}

function broadcastAll(message: WSOutgoingMessage): void {
  for (const [, client] of wsClients) {
    if (client.ws.readyState === WebSocket.OPEN) {
      client.ws.send(JSON.stringify(message));
    }
  }
}

async function start() {
  const httpServer = createServer(async (req, res) => {
    // Convert Node.js request to Web Fetch API
    const { method, url, headers: rawHeaders } = req;

    // Collect body
    const chunks: Buffer[] = [];
    for await (const chunk of req) {
      chunks.push(chunk);
    }
    const body = Buffer.concat(chunks);

    const headers = new Headers();
    for (const [key, value] of Object.entries(rawHeaders)) {
      if (value) headers.set(key, value as string);
    }

    const request = new Request(`http://localhost${url}`, {
      method,
      headers,
      body: method !== "GET" && method !== "HEAD" ? body : undefined,
    });

    const response = await app.fetch(request);

    res.writeHead(response.status, Object.fromEntries(response.headers));
    if (response.body) {
      const reader = response.body.getReader();
      const pump = async () => {
        while (true) {
          const { done, value } = await reader.read();
          if (done) break;
          res.write(value);
        }
        res.end();
      };
      await pump();
    } else {
      res.end();
    }
  });

  // Setup WebSocket on the same HTTP server
  setupWebSocket(httpServer);

  const PORT = Number(process.env.PORT) || 3000;
  httpServer.listen(PORT, () => {
    console.log(`
🚀 Styde Forge running on http://localhost:${PORT}
   Mode: ${MODE}
   SSE:  /sse
   WS:   /ws
   API:  /api/*
    `);
  });
}
```

### 6.2 WebSocket Client Hook (React)

```typescript
// app/hooks/useWebSocket.ts
import { useEffect, useRef, useState, useCallback } from "react";

interface UseWebSocketOptions {
  projectId?: number;
  token?: string;
  enabled?: boolean;
  reconnectInterval?: number;
  maxReconnectAttempts?: number;
}

interface UseWebSocketReturn {
  ws: WebSocket | null;
  connected: boolean;
  lastMessage: WSOutgoingMessage | null;
  sendMessage: (message: WSIncomingMessage) => void;
  subscribe: (projectId: number) => void;
  onMessage: <T extends WSOutgoingMessage["type"]>(
    type: T,
    handler: (payload: Extract<WSOutgoingMessage, { type: T }>["payload"]) => void
  ) => () => void;
}

export function useWebSocket(
  options: UseWebSocketOptions = {}
): UseWebSocketReturn {
  const {
    projectId,
    token,
    enabled = true,
    reconnectInterval = 3_000,
    maxReconnectAttempts = 10,
  } = options;

  const [connected, setConnected] = useState(false);
  const [lastMessage, setLastMessage] = useState<WSOutgoingMessage | null>(null);
  const wsRef = useRef<WebSocket | null>(null);
  const reconnectCount = useRef(0);
  const reconnectTimer = useRef<NodeJS.Timeout>();
  const handlers = useRef<Map<string, Set<(payload: any) => void>>>(new Map());

  // Register a typed message handler
  const onMessage = useCallback(
    <T extends string>(
      type: T,
      handler: (payload: any) => void
    ): (() => void) => {
      if (!handlers.current.has(type)) {
        handlers.current.set(type, new Set());
      }
      handlers.current.get(type)!.add(handler);

      return () => {
        handlers.current.get(type)?.delete(handler);
      };
    },
    []
  );

  // Send a message
  const sendMessage = useCallback((message: WSIncomingMessage) => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      wsRef.current.send(JSON.stringify(message));
    }
  }, []);

  // Subscribe to a project's updates
  const subscribe = useCallback(
    (projectId: number) => {
      sendMessage({ type: "subscribe:project", payload: { projectId } });
    },
    [sendMessage]
  );

  // Connect / reconnect logic
  useEffect(() => {
    if (!enabled || !token) return;

    const connect = () => {
      const params = new URLSearchParams();
      params.set("token", token);
      if (projectId) params.set("projectId", String(projectId));

      const protocol = location.protocol === "https:" ? "wss:" : "ws:";
      const url = `${protocol}//${location.host}/ws?${params.toString()}`;

      const ws = new WebSocket(url);
      wsRef.current = ws;

      ws.addEventListener("open", () => {
        console.log("🔌 WebSocket connected");
        setConnected(true);
        reconnectCount.current = 0;

        // Auto-subscribe to project if provided
        if (projectId) {
          ws.send(
            JSON.stringify({
              type: "subscribe:project",
              payload: { projectId },
            })
          );
        }
      });

      ws.addEventListener("message", (event) => {
        try {
          const message = JSON.parse(event.data) as WSOutgoingMessage;
          setLastMessage(message);

          // Dispatch to registered type handlers
          const typeHandlers = handlers.current.get(message.type);
          if (typeHandlers) {
            for (const handler of typeHandlers) {
              handler(message.payload);
            }
          }

          // Pong handling
          if (message.type === "pong") {
            // Could update latency metrics here
          }
        } catch (err) {
          console.error("Failed to parse WebSocket message:", err);
        }
      });

      ws.addEventListener("close", (event) => {
        console.log(`🔌 WebSocket closed: ${event.code} ${event.reason}`);
        setConnected(false);

        // Reconnect with backoff (unless explicitly closed with 4001 = Unauthorized)
        if (
          event.code !== 4001 &&
          reconnectCount.current < maxReconnectAttempts
        ) {
          const delay = reconnectInterval * Math.pow(1.5, reconnectCount.current);
          console.log(`Reconnecting in ${delay}ms (attempt ${reconnectCount.current + 1})`);
          reconnectTimer.current = setTimeout(connect, delay);
          reconnectCount.current++;
        }
      });

      ws.addEventListener("error", (err) => {
        console.error("WebSocket error:", err);
        // close event will fire after this
      });
    };

    connect();

    return () => {
      clearTimeout(reconnectTimer.current);
      wsRef.current?.close();
      wsRef.current = null;
      setConnected(false);
    };
  }, [enabled, token, projectId, reconnectInterval, maxReconnectAttempts]);

  // Ping every 30 seconds to keep connection alive
  useEffect(() => {
    if (!connected) return;

    const pingInterval = setInterval(() => {
      sendMessage({ type: "ping" });
    }, 30_000);

    return () => clearInterval(pingInterval);
  }, [connected, sendMessage]);

  return {
    ws: wsRef.current,
    connected,
    lastMessage,
    sendMessage,
    subscribe,
    onMessage,
  };
}
```

### 6.4 WebSocket Protocol Reference

| Direction | Type | Payload | Purpose |
|---|---|---|---|
| **Client → Server** | `ping` | `{}` | Keepalive |
| **Server → Client** | `pong` | `{ timestamp }` | Keepalive response |
| **Client → Server** | `subscribe:project` | `{ projectId }` | Join a project room |
| **Server → Client** | `subscribed` | `{ projectId }` | Confirmation |
| **Client → Server** | `cursor:move` | `{ x, y }` | Broadcast cursor position |
| **Server → Client** | `cursor:move` | `{ userId, x, y }` | Remote cursor update |
| **Server → Client** | `user:online` | `{ userId, projectId }` | User joined project |
| **Server → Client** | `user:offline` | `{ userId }` | User left project |
| **Client → Server** | `comment:add` | `{ body, authorName }` | Add a comment |
| **Server → Client** | `comment:add` | `{ id, body, authorId, authorName, timestamp }` | New comment |
| **Client → Server** | `project:update` | `{ field, value }` | Collaborative edit |

```typescript
app.all("*", createRequestHandler({ build }));

// Start server
const httpServer = createServer(app.fetch);
const wss = new WebSocketServer({ server: httpServer, path: "/ws" });

let isShuttingDown = false;

// Graceful shutdown
process.on("SIGTERM", async () => {
  console.log("SIGTERM received — shutting down gracefully");
  isShuttingDown = true;

  // Wait for active connections to drain (max 25s)
  await new Promise((resolve) => setTimeout(resolve, 25_000));

  wss.close();
  httpServer.close(() => {
    console.log("Server closed");
    process.exit(0);
  });
});

httpServer.listen(Number(process.env.PORT) || 3000);
```

### 9.4 Performance Characteristics

| Pattern | Time to Interactive | Network | Best For |
|---|---|---|---|
| **SPA Mode** | ~1.5s (initial) / ~0ms (subsequent nav) | 1 initial bundle | Dashboards, internal tools |
| **clientLoader** | ~0ms (cache hit) | 0 (cached) | Back/forward navigation |
| **SSE** | Immediate + streaming | 1 persistent connection | Activity feeds, notifications |
| **defer/Suspense** | ~100ms (shell) + streaming | Server-rendered shell | Data-heavy pages |
| **WebSocket** | Real-time (<50ms) | 1 persistent connection | Collaboration, live cursors |

---

## Key Takeaways

1. **SPA Mode** turns Remix into a full client-side React app — use `ssr: false` in Vite config + `HydrateFallback` in root. No server loaders needed.

2. **clientLoader** with an LRU cache eliminates network waterfalls for repeat navigation. Always call `serverLoader()` in the background for freshness.

3. **SSE** provides a simple, HTTP-native real-time stream. Use a global `EventEmitter` on the server and `EventSource` on the client. Perfect for dashboards and activity feeds.

4. **defer/Suspense** streams page content: critical data in the initial HTML shell, slow data streamed progressively. Each deferred promise gets its own `<Suspense>` boundary with `Await` and `errorElement`.

5. **Custom server with WebSocket** using Hono + ws enables bi-directional real-time collaboration on the same HTTP port. Authenticate on connect, broadcast per-project, and handle graceful reconnection.

6. All five patterns compose cleanly: SPA mode for the shell, clientLoader for instant nav, SSE for live dashboards, defer for detailed pages, and WebSocket for real-time collaboration.
