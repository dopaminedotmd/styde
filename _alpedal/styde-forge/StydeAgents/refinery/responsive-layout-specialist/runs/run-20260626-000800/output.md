## Key Breakpoints & Behavior Summary

### 12-Column Grid
| Viewport | Column Behavior |
|----------|----------------|
| ≥ 64rem (1024px) | All 12 columns independent |
| 40rem–64rem (640–1024px) | col-8+ → full width, col-4/5 → col-6, col-3 → col-4 |
| < 40rem (640px) | All columns → full width, except col-2 → col-6 |

### Container Query Cards
| Container Width | Card Layout |
|----------------|-------------|
| < 250px | Image hidden, compact text, minimal padding |
| 250–400px | Small image, 2-line text clamp |
| 400–600px | Standard card, 3-line clamp |
| ≥ 600px | Horizontal layout (image left, content right) |

### Fluid Typography Scale (modular ratio ~1.2)
| Token | Range | Usage |
|-------|-------|-------|
| `--fs-6` | 2.488rem → 2.986rem | H1 |
| `--fs-5` | 2.074rem → 2.488rem | H2 |
| `--fs-4` | 1.728rem → 2.074rem | H3 |
| `--fs-3` | 1.44rem → 1.728rem | H4 |
| `--fs-2` | 1.2rem → 1.44rem | H5 / Card titles |
| `--fs-1` | 1rem → 1.2rem | H6 |
| `--fs-0` | 0.833rem → 1rem | Body |
| `--fs--1` | 0.694rem → 0.833rem | Small / Captions |
| `--fs--2` | 0.579rem → 0.694rem | Badges / Fine print |

### Sidebar → Bottom Nav Transition
- **Desktop (≥ 48rem / 768px):** Sticky sidebar with full nav, user profile, and sections
- **Mobile (< 48rem):** Sidebar hidden; fixed bottom nav with 5 icon+label items; hamburger menu opens slide-out overlay sidebar

### Data Table → Cards
- **Container ≥ 768px:** Full table with all columns
- **Container 560–768px:** Priority-3 columns hidden
- **Container < 560px:** Table transforms to stacked cards with `data-label` pseudo-elements as labels

## Browser Support

- CSS Grid: All modern browsers (Chrome 57+, Firefox 52+, Safari 10.1+, Edge 16+)
- Container Queries: Chrome 105+, Safari 16+, Firefox 110+, Edge 105+
- `clamp()`: Chrome 79+, Firefox 75+, Safari 13.1+, Edge 79+
- `@container` fallback: Cards gracefully fall back to standard block layout in non-supporting browsers (container queries are progressive enhancement)

Save the HTML block above as an `.html` file and open in any modern browser. All features are self-contained — no external dependencies, no build step required.
