## 1. Advanced System Overview

The **Styde Forge Advanced Animation System (Cycle 2)** adds four high-power animation primitives on top of the Cycle 1 foundation. These target modern browser APIs and deliver motion that responds to scroll position, page navigation, layout reflow, and touch/pointer gestures. Every module is self-contained, zero-dependency, and progressively enhanced.

### Design Principles (extended)

| Principle | Description |
|-----------|-------------|
| **Zero-dependency** | Pure CSS + vanilla JS. No frameworks, no polyfill shims in core. |
| **Progressive enhancement** | Native APIs where supported; graceful fallbacks otherwise. |
| **GPU-accelerated** | `transform` + `opacity` only in animation property sets. |
| **CSS custom properties** | All timing, distance, and threshold values configurable via variables. |
| **Scroll-driven native** | Uses CSS `scroll-timeline` / `view-timeline` (with JS fallback). |
| **FLIP-first layout** | First-Last-Invert-Play ensures 60fps layout-shift animations. |
| **Multi-touch gesture** | Pointer events API for pan, swipe, pinch, and rotate gestures. |

## 2. Scroll-Driven Animations — ScrollTimeline

**Files:** `css/scroll-timeline.css` + `js/scroll-timeline.js`

Drives animations directly from scroll position using the native **CSS `scroll-timeline`** (with `animation-timeline`) in Chromium browsers, backed by a high-performance JS shim for Firefox and Safari. Supports both **scroll progress timelines** (linked to a scroll container's offset) and **view progress timelines** (linked to an element's intersection with its scrollport).

### 2.2 JavaScript — ScrollTimeline Engine

```javascript
/**
 * scroll-timeline.js
 * Styde Forge Advanced Animation System — ScrollTimeline Engine
 *
 * Provides a JS-powered fallback for browsers that do not support
 * CSS scroll-timeline / view-timeline (Firefox, Safari < 18).
 *
 * When native support is detected the JS driver does nothing —
 * the CSS declarations above handle everything. On unsupported
 * browsers, a lightweight IntersectionObserver + scroll listener
 * drives the same CSS custom properties.
 *
 * Architecture:
 *   - Feature-detect `CSS.supports('animation-timeline: scroll()')`
 *   - If unsupported: map scroll/view progress to
 *     `--st-scroll-progress` (0–1) and `--st-view-progress` (0–1)
 *     custom properties on each target.
 *   - CSS uses those properties in calc()-based animation keyframes.
 *
 * Usage:
 *   <div class="st-scroll-container">
 *     <div class="st-scroll-driven st-scroll-fx-progress"
 *          data-st-scroll-drive="scroll"> ... </div>
 *   </div>
 *
 *   <div class="st-view-driven st-view-reveal"
 *        data-st-scroll-drive="view"> ... </div>
 */

(function () {
  'use strict';

  /**
   * Feature detection for native scroll-driven animations.
   * Checks both scroll() and view() timeline support.
   */
  function supportsScrollTimeline() {
    try {
      return CSS.supports('animation-timeline: scroll()');
    } catch (e) {
      return false;
    }
  }

  /**
   * Map a value from one range to another.
   */
  function mapRange(value, inMin, inMax, outMin, outMax) {
    if (inMax === inMin) return outMin;
    const t = (value - inMin) / (inMax - inMin);
    return outMin + t * (outMax - outMin);
  }

  /**
   * Clamp a value between min and max.
   */
  function clamp(value, min, max) {
    return Math.min(Math.max(value, min), max);
  }

  /**
   * Calculate scroll progress for a scroll-driven element.
   * Progress is 0 when the scroll container is at its start,
   * and 1 when it reaches the end of the scrollable area.
   */
  function computeScrollProgress(element) {
    const container = element.closest('.st-scroll-container') || element.parentElement;
    const scrollTop = container.scrollTop;
    const maxScroll = container.scrollHeight - container.clientHeight;

    if (maxScroll <= 0) return 0;
    return clamp(scrollTop / maxScroll, 0, 1);
  }

  /**
   * Calculate view progress for a view-driven element.
   * Progress is 0 when the element first enters the viewport
   * and 1 when it fully exits.
   */
  function computeViewProgress(element) {
    const rect = element.getBoundingClientRect();
    const windowHeight = window.innerHeight;

    // The element's vertical range relative to the viewport:
    // Top of element relative to viewport bottom (entry point)
    // Bottom of element relative to viewport top (exit point)
    const entryPoint = windowHeight;        // element top at viewport bottom
    const exitPoint = -rect.height;         // element bottom at viewport top

    const position = rect.top;

    if (position > entryPoint) return 0;
    if (position < exitPoint) return 1;

    return clamp(mapRange(position, entryPoint, exitPoint, 0, 1), 0, 1);
  }

  /**
   * Update all scroll-driven and view-driven elements.
   */
  function updateAllProgress() {
    const scrollDriven = document.querySelectorAll(
      '[data-st-scroll-drive="scroll"]'
    );
    const viewDriven = document.querySelectorAll(
      '[data-st-scroll-drive="view"]'
    );

    scrollDriven.forEach(function (el) {
      const progress = computeScrollProgress(el);
      el.style.setProperty('--st-scroll-progress', progress.toFixed(4));
    });

    viewDriven.forEach(function (el) {
      const progress = computeViewProgress(el);
      el.style.setProperty('--st-view-progress', progress.toFixed(4));
    });
  }

  /**
   * Set up scroll listeners on all .st-scroll-container elements.
   * Uses passive listeners for performance.
   */
  function bindScrollContainers() {
    const containers = document.querySelectorAll('.st-scroll-container');
    containers.forEach(function (container) {
      if (container.dataset.stScrollBound === 'true') return;
      container.dataset.stScrollBound = 'true';

      container.addEventListener('scroll', updateAllProgress, {
        passive: true,
      });
    });

    // Also listen on window for view-driven animations
    if (!window._stScrollWindowBound) {
      window._stScrollWindowBound = true;
      window.addEventListener('scroll', updateAllProgress, { passive: true });
      window.addEventListener('resize', updateAllProgress, { passive: true });
    }
  }

  /**
   * Bootstrap: if native scroll-timeline is supported, do nothing.
   * Otherwise, initialise the JS shim.
   */
  function init() {
    if (supportsScrollTimeline()) {
      console.log(
        '[StydeForge] Native scroll-timeline supported. JS shim idle.'
      );
      return;
    }

    console.log(
      '[StydeForge] Scroll-timeline NOT supported. Activating JS shim.'
    );

    bindScrollContainers();
    updateAllProgress();

    // Re-bind after potential DOM mutations
    const mutationObserver = new MutationObserver(function () {
      bindScrollContainers();
      updateAllProgress();
    });

    mutationObserver.observe(document.body, {
      childList: true,
      subtree: true,
    });
  }

  // --- Bootstrap ---
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', init);
  } else {
    init();
  }

  /**
   * Public API.
   */
  window.StydeScrollTimeline = {
    supportsScrollTimeline: supportsScrollTimeline,
    refresh: updateAllProgress,
  };
})();
```

### 3.2 JavaScript — SPA View Transition Engine

```javascript
/**
 * view-transitions.js
 * Styde Forge Advanced Animation System — SPA View Transition Engine
 *
 * Provides SPA navigation with the View Transitions API.
 * Features:
 *   - Feature detection for document.startViewTransition
 *   - Direction-aware transitions (forward, back, modal)
 *   - History-aware: back/forward navigation reverses direction
 *   - Named element morphing via view-transition-name
 *   - Loading bar during async transitions
 *   - Fallback to CSS overlay transitions for unsupported browsers
 *   - Event hooks: st-vt:start, st-vt:complete, st-vt:error
 *
 * Usage:
 *   StydeViewTransitions.navigate('/about', { type: 'forward' });
 *   <a href="/about" data-st-vt="forward">About</a>
 */

(function () {
  'use strict';

  const DURATION = 400; // ms

  /**
   * Check native View Transitions API support.
   */
  function supportsViewTransitions() {
    return (
      typeof document !== 'undefined' &&
      typeof document.startViewTransition === 'function'
    );
  }

  /**
   * Navigation history stack for direction detection.
   */
  const historyStack = [window.location.pathname];
  let navigationDirection = 'forward';

  /**
   * Push a new URL onto the history stack.
   */
  function pushHistory(url) {
    historyStack.push(url);
    while (historyStack.length > 50) {
      historyStack.shift();
    }
    navigationDirection = 'forward';
  }

  /**
   * Determine direction from history stack.
   */
  function getDirection(url) {
    const idx = historyStack.lastIndexOf(url);
    if (idx >= 0 && idx < historyStack.length - 1) {
      navigationDirection = 'back';
    }
    return navigationDirection;
  }

  /**
   * Set the transition direction class on <html>.
   */
  function setTransitionClass(type, isBack) {
    const html = document.documentElement;
    html.classList.remove(
      'st-vt-dir-forward',
      'st-vt-dir-back',
      'st-vt-dir-modal'
    );

    if (isBack && type === 'forward') {
      html.classList.add('st-vt-dir-back');
    } else if (type === 'forward') {
      html.classList.add('st-vt-dir-forward');
    } else if (type === 'back') {
      html.classList.add('st-vt-dir-back');
    } else if (type === 'modal') {
      html.classList.add('st-vt-dir-modal');
    }
  }

  /**
   * Show / hide the loading bar.
   */
  function showLoadingBar() {
    let bar = document.getElementById('st-vt-loading-bar');
    if (!bar) {
      bar = document.createElement('div');
      bar.id = 'st-vt-loading-bar';
      bar.className = 'st-vt-loading-bar';
      document.body.appendChild(bar);
    }
    bar.classList.add('active');
    return bar;
  }

  function hideLoadingBar(bar) {
    if (!bar) return;
    bar.classList.remove('active');
    // Reset animation by forcing reflow
    void bar.offsetWidth;
    bar.style.transform = 'scaleX(0)';
  }

  /**
   * Fetch a URL and return its full HTML string.
   */
  async function fetchPage(url) {
    const response = await fetch(url, {
      headers: { 'X-Requested-With': 'XMLHttpRequest' },
    });

    if (!response.ok) {
      throw new Error(
        `Failed to fetch ${url}: ${response.status} ${response.statusText}`
      );
    }

    return response.text();
  }

  /**
   * Apply new page HTML into the current document.
   * Merges <head> styles/scripts and replaces <body>.
   * Preserves scripts marked with data-st-vt-persist.
   */
  function applyPageHTML(html) {
    const parser = new DOMParser();
    const doc = parser.parseFromString(html, 'text/html');

    // Update title
    if (doc.title) {
      document.title = doc.title;
    }

    // Merge <head> elements
    const existingHeadIds = new Set(
      Array.from(document.head.querySelectorAll('[id]')).map(function (el) {
        return el.id;
      })
    );

    const newHead = doc.querySelector('head');
    if (newHead) {
      newHead.querySelectorAll('link[rel="stylesheet"], script[src]').forEach(
        function (el) {
          const id = el.getAttribute('id');
          if (id && existingHeadIds.has(id)) return;
          document.head.appendChild(el.cloneNode(true));
        }
      );
    }

    // Replace <body> content
    const newBody = doc.querySelector('body');
    if (newBody) {
      document.body.innerHTML = newBody.innerHTML;
    }

    // Dispatch content-updated event for re-initialization
    window.dispatchEvent(
      new CustomEvent('st-vt:content-updated', {
        detail: { url: window.location.pathname },
      })
    );
  }

  /**
   * Fallback: CSS overlay transition for browsers without VT API.
   */
  async function fallbackTransition(url, type) {
    const bar = showLoadingBar();

    // Phase 1: Fade out
    await animateOverlay('out', DURATION);

    // Phase 2: Fetch + swap
    const html = await fetchPage(url);
    applyPageHTML(html);
    window.history.pushState({ vtType: type }, '', url);

    // Phase 3: Fade in
    await animateOverlay('in', DURATION);

    hideLoadingBar(bar);
  }

  /**
   * Simple overlay animation for fallback.
   */
  function animateOverlay(direction, duration) {
    return new Promise(function (resolve) {
      let overlay = document.getElementById('st-vt-overlay');
      if (!overlay) {
        overlay = document.createElement('div');
        overlay.id = 'st-vt-overlay';
        overlay.style.cssText =
          'position:fixed;inset:0;z-index:9999;' +
          'background:var(--st-vt-overlay-bg,#0f172a);' +
          'opacity:0;pointer-events:none;' +
          'transition:opacity ' + duration + 'ms ease;';
        document.body.appendChild(overlay);
      }

      if (direction === 'out') {
        overlay.style.opacity = '1';
      } else {
        overlay.style.opacity = '0';
      }

      setTimeout(function () {
        if (direction === 'in') {
          overlay.style.opacity = '0';
        }
        resolve();
      }, duration);
    });
  }

  /**
   * Native View Transitions API transition.
   */
  async function vtTransition(url, type, isBack) {
    const bar = showLoadingBar();

    setTransitionClass(type, isBack);

    const html = await fetchPage(url);

    // Use double rAF to ensure the class takes effect before snapshot
    await new Promise(function (resolve) {
      requestAnimationFrame(function () {
        requestAnimationFrame(resolve);
      });
    });

    const transition = document.startViewTransition(function () {
      applyPageHTML(html);
    });

    try {
      await transition.finished;
    } catch (err) {
      // Transition was skipped (e.g., due to render-blocking)
      console.warn('[StydeForge] View transition skipped:', err.message);
    }

    hideLoadingBar(bar);

    // Clean up direction classes
    document.documentElement.classList.remove(
      'st-vt-dir-forward',
      'st-vt-dir-back',
      'st-vt-dir-modal'
    );
  }

  /**
   * Main navigate function.
   *
   * @param {string} url        - Target URL
   * @param {object} [options]  - { type: 'forward' | 'back' | 'modal' }
   */
  async function navigate(url, options) {
    options = options || {};
    var type = options.type || 'forward';
    var isBack = getDirection(url) === 'back';

    // Dispatch start event
    window.dispatchEvent(
      new CustomEvent('st-vt:start', {
        detail: { url: url, type: type, direction: isBack ? 'back' : 'forward' },
      })
    );

    try {
      if (supportsViewTransitions()) {
        await vtTransition(url, type, isBack);
      } else {
        await fallbackTransition(url, type);
      }

      pushHistory(url);

      // Re-initialise other Styde modules
      if (window.StydeScrollTimeline && window.StydeScrollTimeline.refresh) {
        window.StydeScrollTimeline.refresh();
      }

      // Dispatch complete event
      window.dispatchEvent(
        new CustomEvent('st-vt:complete', {
          detail: { url: url, type: type },
        })
      );
    } catch (err) {
      console.error('[StydeForge] Navigation error:', err);

      // Dispatch error event
      window.dispatchEvent(
        new CustomEvent('st-vt:error', {
          detail: { url: url, error: err.message },
        })
      );

      // Navigate the traditional way as last resort
      window.location.href = url;
    }
  }

  /**
   * Intercept link clicks for SPA navigation.
   * Opt-in: add data-st-vt="forward|back|modal" to <a> tags.
   */
  function initLinkInterceptor() {
    document.addEventListener('click', function (e) {
      var link = e.target.closest('a[data-st-vt]');
      if (!link) return;

      var href = link.getAttribute('href');
      if (!href) return;

      // Skip external links and anchors
      if (
        href.startsWith('http') ||
        href.startsWith('//') ||
        href.startsWith('#') ||
        href.startsWith('mailto:') ||
        href.startsWith('tel:')
      ) {
        return;
      }

      // Skip if modifier keys are held
      if (e.metaKey || e.ctrlKey || e.shiftKey || e.altKey) return;

      e.preventDefault();

      var type = link.getAttribute('data-st-vt') || 'forward';
      navigate(href, { type: type });
    });
  }

  /**
   * Handle browser back/forward buttons using the View Transitions API.
   */
  function initPopStateHandler() {
    window.addEventListener('popstate', function (e) {
      var url = window.location.pathname;
      var type = e.state && e.state.vtType ? e.state.vtType : 'back';

      if (supportsViewTransitions()) {
        // Re-fetch and re-apply for consistent transitions
        fetchPage(url)
          .then(function (html) {
            document.documentElement.classList.add('st-vt-dir-back');

            var transition = document.startViewTransition(function () {
              applyPageHTML(html);
            });

            return transition.finished;
          })
          .then(function () {
            document.documentElement.classList.remove('st-vt-dir-back');
            window.dispatchEvent(
              new CustomEvent('st-vt:complete', {
                detail: { url: url, type: 'back' },
              })
            );
          })
          .catch(function () {
            // Transition skipped, no-op
          });
      }
    });
  }

  // --- Bootstrap ---
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', function () {
      initLinkInterceptor();
      initPopStateHandler();
    });
  } else {
    initLinkInterceptor();
    initPopStateHandler();
  }

  /**
   * Public API.
   */
  window.StydeViewTransitions = {
    navigate: navigate,
    supportsViewTransitions: supportsViewTransitions,
  };
})();
```

## 4. FLIP Animations — Layout Shifts

**File:** `js/flip.js`

The **FLIP technique** (First, Last, Invert, Play) animates elements smoothly from one layout position to another when the DOM changes — grid reordering, flex-wrap shifts, filter/sort transitions, and content insertion. This module captures element bounding rects before a layout change, calculates the difference, inverts the transform to make the element appear in its old position, then animates it to the new position using a single `requestAnimationFrame`-driven transition.

### 4.1 JavaScript — FLIP Animation Engine

```javascript
/**
 * flip.js
 * Styde Forge Advanced Animation System — FLIP Animation Engine
 *
 * Captures First and Last positions of elements, Inverts the transform,
 * and Plays the animation for smooth layout transitions.
 *
 * Features:
 *   - Group-based FLIP: animate multiple elements at once
 *   - Stagger support: delay each element by an index-based offset
 *   - Custom easings via CSS custom properties
 *   - Nested FLIP: animate parent and children independently
 *   - Auto-detection: watches for DOM mutations via MutationObserver
 *   - Cancel-safe: active transitions are cancelled before new ones start
 *
 * Usage:
 *   const flip = StydeFLIP.capture('.grid-item');
 *   // ... change layout (e.g., reorder, filter, resize) ...
 *   flip.animate({ stagger: 50, easing: 'spring' });
 *
 *   // Or declarative:
 *   <div class="st-flip-group" data-st-flip-stagger="60">
 *     <div class="st-flip-item">Item 1</div>
 *     <div class="st-flip-item">Item 2</div>
 *   </div>
 */

(function () {
  'use strict';

  /**
   * Named easings available via string shortcuts.
   */
  var EASINGS = {
    ease: 'cubic-bezier(0.4, 0, 0.2, 1)',
    'ease-in': 'cubic-bezier(0.4, 0, 1, 1)',
    'ease-out': 'cubic-bezier(0, 0, 0.2, 1)',
    'ease-in-out': 'cubic-bezier(0.4, 0, 0.2, 1)',
    spring: 'cubic-bezier(0.34, 1.56, 0.64, 1)',
    bounce: 'cubic-bezier(0.68, -0.55, 0.265, 1.55)',
    'anticipate': 'cubic-bezier(0.68, -0.1, 0.265, 0.9)',
  };

  /**
   * Snapshot: capture the current bounding rect of every element
   * in a NodeList or single element.
   */
  function capture(elements) {
    if (!elements) return null;

    var items = [];
    var list = elements.length !== undefined ? elements : [elements];

    for (var i = 0; i < list.length; i++) {
      var el = list[i];
      if (!el || !el.getBoundingClientRect) continue;

      var rect = el.getBoundingClientRect();
      var computed = window.getComputedStyle(el);

      items.push({
        element: el,
        rect: {
          top: rect.top,
          left: rect.left,
          width: rect.width,
          height: rect.height,
        },
        opacity: parseFloat(computed.opacity),
        transform: computed.transform,
        transition: computed.transition,
        borderRadius: computed.borderRadius,
        // Snapshot children for nested FLIP
        children: el.children.length > 0 ? capture(el.children) : null,
      });
    }

    return { items: items };
  }

  /**
   * Animate from the captured snapshot to the current layout.
   *
   * @param {object} snapshot   - Result from capture()
   * @param {object} [options]  - { stagger, easing, duration, scale, opacity }
   */
  function animate(snapshot, options) {
    if (!snapshot || !snapshot.items) return Promise.resolve();

    options = options || {};
    var stagger = options.stagger || 0;
    var easing = EASINGS[options.easing] || options.easing ||
      'cubic-bezier(0.4, 0, 0.2, 1)';
    var duration = options.duration || 400;
    var animateScale = options.scale !== false;
    var animateOpacity = options.opacity !== false;

    var promises = [];

    snapshot.items.forEach(function (item, index) {
      var el = item.element;
      if (!el || !el.getBoundingClientRect) return;

      var first = item.rect;
      var last = el.getBoundingClientRect();

      // Calculate the difference (invert)
      var dx = first.left - last.left;
      var dy = first.top - last.top;
      var dw = first.width / (last.width || 1);
      var dh = first.height / (last.height || 1);

      // If nothing changed, skip
      if (dx === 0 && dy === 0 && dw === 1 && dh === 1) return;

      // Capture current transition and remove it
      var prevTransition = el.style.transition;

      // Invert: position the element where it was
      el.style.transition = 'none';
      el.style.transform =
        'translate(' + dx + 'px, ' + dy + 'px)' +
        (animateScale ? ' scale(' + dw + ', ' + dh + ')' : '');

      if (animateOpacity && item.opacity !== undefined) {
        el.style.opacity = item.opacity;
      }

      // Force layout recalculation
      void el.offsetWidth;

      // Play: transition to the new (actual) position
      var delay = index * stagger;

      el.style.transition =
        'transform ' + duration + 'ms ' + easing + ' ' + delay + 'ms' +
        (animateOpacity ? ', opacity ' + duration + 'ms ' + easing + ' ' + delay + 'ms' : '');
      el.style.transform = 'translate(0, 0) scale(1, 1)';
      el.style.opacity = '';

      // Animate nested children
      if (item.children && item.children.items.length > 0) {
        var childPromise = animate(item.children, {
          stagger: stagger,
          easing: easing,
          duration: duration,
          scale: animateScale,
          opacity: animateOpacity,
        });
        promises.push(childPromise);
      }

      // Track this element's animation completion
      var promise = new Promise(function (resolve) {
        var handler = function (e) {
          if (e.target !== el) return;
          el.removeEventListener('transitionend', handler);
          el.style.transition = prevTransition;
          el.style.transform = '';
          resolve();
        };
        el.addEventListener('transitionend', handler);

        // Fallback: resolve after duration + delay + 20ms
        setTimeout(function () {
          el.removeEventListener('transitionend', handler);
          el.style.transition = prevTransition;
          el.style.transform = '';
          resolve();
        }, duration + delay + 20);
      });

      promises.push(promise);
    });

    return Promise.all(promises);
  }

  /**
   * Convenience: capture + animate in one call.
   */
  function flip(elements, changeFn, options) {
    var snapshot = capture(elements);

    // Execute the layout change
    if (typeof changeFn === 'function') {
      changeFn();
    }

    // Wait a tick for layout to settle
    return new Promise(function (resolve) {
      requestAnimationFrame(function () {
        requestAnimationFrame(function () {
          animate(snapshot, options).then(resolve);
        });
      });
    });
  }

  /**
   * Declarative FLIP: watch for layout changes on .st-flip-group
   * and animate .st-flip-item children automatically.
   */
  function initDeclarativeFLIP() {
    var groups = document.querySelectorAll('.st-flip-group');

    groups.forEach(function (group) {
      if (group.dataset.stFlipInitialised === 'true') return;
      group.dataset.stFlipInitialised = 'true';

      var stagger = parseInt(group.getAttribute('data-st-flip-stagger'), 10) || 0;
      var easing = group.getAttribute('data-st-flip-easing') || 'ease';
      var duration = parseInt(group.getAttribute('data-st-flip-duration'), 10) || 400;

      // Watch for child list changes
      var observer = new MutationObserver(function (mutations) {
        mutations.forEach(function (mutation) {
          if (mutation.type !== 'childList') return;

          var items = group.querySelectorAll('.st-flip-item');
          var snapshot = capture(items);

          // Wait for layout
          requestAnimationFrame(function () {
            requestAnimationFrame(function () {
              animate(snapshot, {
                stagger: stagger,
                easing: easing,
                duration: duration,
              });
            });
          });
        });
      });

      observer.observe(group, { childList: true });
    });
  }

  // --- Bootstrap ---
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initDeclarativeFLIP);
  } else {
    initDeclarativeFLIP();
  }

  /**
   * Re-initialise for dynamically added content.
   */
  function refresh() {
    var groups = document.querySelectorAll('.st-flip-group');
    groups.forEach(function (group) {
      group.dataset.stFlipInitialised = 'false';
    });
    initDeclarativeFLIP();
  }

  /**
   * Public API.
   */
  window.StydeFLIP = {
    capture: capture,
    animate: animate,
    flip: flip,
    refresh: refresh,
  };
})();
```

## 5. Gesture-Driven Animations

**Files:** `css/gesture.css` + `js/gesture-engine.js`

A lightweight, zero-dependency gesture recognition system built on **Pointer Events**. ### 5.2 JavaScript — Gesture Engine

```javascript
/**
 * gesture-engine.js
 * Styde Forge Advanced Animation System — Gesture Engine
 *
 * Recognises touch/pointer gestures on elements marked with
 * [data-st-gesture] and emits custom DOM events with rich detail.
 * Updates CSS custom properties in real time for CSS-driven animations.
 *
 * Supported gestures:
 *   pan    — Single-pointer drag: deltaX, deltaY, velocityX, velocityY
 *   swipe  — Velocity-threshold flick: direction, velocity
 *   pinch  — Two-pointer scale: scale, originX, originY
 *   rotate — Two-pointer rotation: angle (degrees)
 *   tilt   — Pointer position relative to element centre
 *
 * Events emitted (on the gesture target element):
 *   st:gesture:start   — gesture begins
 *   st:gesture:move    — real-time movement (throttled to ~60fps)
 *   st:gesture:end     — gesture ends
 *   st:pan:start       — pan gesture begins
 *   st:pan:move        — pan in progress
 *   st:pan:end         — pan gesture ends
 *   st:swipe:left      — directional swipe (left/right/up/down)
 *   st:swipe:right
 *   st:swipe:up
 *   st:swipe:down
 *   st:pinch:start     — pinch gesture begins
 *   st:pinch:move      — pinch in progress
 *   st:pinch:end       — pinch gesture ends
 *   st:rotate:start    — rotate gesture begins
 *   st:rotate:move     — rotate in progress
 *   st:rotate:end      — rotate gesture ends
 *   st:tilt:move       — tilt updated
 *
 * CSS custom properties set on the target element:
 *   --st-pan-x, --st-pan-y
 *   --st-pan-velocity-x, --st-pan-velocity-y
 *   --st-pinch-scale
 *   --st-pinch-origin-x, --st-pinch-origin-y
 *   --st-rotate-angle
 *   --st-tilt-rotate-x, --st-tilt-rotate-y
 */

(function () {
  'use strict';

  // --- Constants ---
  var SWIPE_VELOCITY_THRESHOLD = 0.3; // px/ms minimum for swipe
  var PINCH_THRESHOLD = 10;           // px minimum separation change for pinch
  var TAP_MAX_MOVEMENT = 10;          // px max movement for it to be a tap
  var TAP_MAX_DURATION = 300;         // ms max duration for it to be a tap

  // --- Utility functions ---

  /**
   * Calculate distance between two points.
   */
  function distance(p1, p2) {
    var dx = p2.clientX - p1.clientX;
    var dy = p2.clientY - p1.clientY;
    return Math.sqrt(dx * dx + dy * dy);
  }

  /**
   * Calculate angle between two points (degrees).
   */
  function angle(p1, p2) {
    return Math.atan2(p2.clientY - p1.clientY, p2.clientX - p1.clientX) *
      (180 / Math.PI);
  }

  /**
   * Calculate midpoint between two points.
   */
  function midpoint(p1, p2) {
    return {
      x: (p1.clientX + p2.clientX) / 2,
      y: (p1.clientY + p2.clientY) / 2,
    };
  }

  /**
   * Emit a custom event on an element with detail data.
   */
  function emit(el, name, detail) {
    el.dispatchEvent(
      new CustomEvent(name, {
        bubbles: true,
        cancelable: true,
        detail: detail,
      })
    );
  }

  /**
   * Set a CSS custom property on an element.
   */
  function setCSSVar(el, name, value) {
    el.style.setProperty(name, value);
  }

  /**
   * Get all gesture types enabled on an element.
   */
  function getEnabledGestures(el) {
    var type = el.getAttribute('data-st-gesture') || 'all';
    if (type === 'all') return ['pan', 'swipe', 'pinch', 'rotate', 'tilt'];
    return type.split(/\s+/).filter(Boolean);
  }

  // --- Gesture State Machine ---

  /**
   * Create a gesture tracker for a single element.
   */
  function createGestureTracker(el) {
    var state = {
      active: false,
      pointers: {},          // active pointer IDs → { clientX, clientY, time }
      startTime: 0,
      startPoint: null,
      lastPoint: null,
      startDistance: 0,
      startAngle: 0,
      totalDeltaX: 0,
      totalDeltaY: 0,
      velocityX: 0,
      velocityY: 0,
      enabledGestures: getEnabledGestures(el),
    };

    var gestureType = null;          // 'pan' | 'pinch' | 'rotate'
    var pointerCount = 0;

    /**
     * Determine gesture type based on pointer count.
     */
    function classifyGesture(count) {
      if (count === 1) {
        if (state.enabledGestures.indexOf('pan') !== -1) return 'pan';
        if (state.enabledGestures.indexOf('swipe') !== -1) return 'pan';
      }
      if (count === 2) {
        if (state.enabledGestures.indexOf('pinch') !== -1 &&
            state.enabledGestures.indexOf('rotate') !== -1) {
          return 'pinch-rotate';
        }
        if (state.enabledGestures.indexOf('pinch') !== -1) return 'pinch';
        if (state.enabledGestures.indexOf('rotate') !== -1) return 'rotate';
      }
      return null;
    }

    /**
     * Get current pointer positions as an array.
     */
    function getActivePointers() {
      return Object.keys(state.pointers).map(function (id) {
        return state.pointers[id];
      });
    }

    // --- Pointer Event Handlers ---

    function onPointerDown(e) {
      state.pointers[e.pointerId] = {
        clientX: e.clientX,
        clientY: e.clientY,
        time: Date.now(),
      };

      pointerCount = Object.keys(state.pointers).length;
      var gesture = classifyGesture(pointerCount);

      if (!gesture || (state.active && gesture !== gestureType)) return;

      // Record start state
      var pointers = getActivePointers();
      state.startTime = Date.now();
      state.startPoint = { x: e.clientX, y: e.clientY };
      state.lastPoint = { x: e.clientX, y: e.clientY };
      state.totalDeltaX = 0;
      state.totalDeltaY = 0;
      state.velocityX = 0;
      state.velocityY = 0;

      if (pointers.length === 2) {
        state.startDistance = distance(pointers[0], pointers[1]);
        state.startAngle = angle(pointers[0], pointers[1]);
      }

      state.active = true;
      gestureType = gesture;

      el.classList.add('st-gesture-active');
      el.setPointerCapture(e.pointerId);

      // Emit start events
      emit(el, 'st:gesture:start', {
        type: gestureType,
        x: e.clientX,
        y: e.clientY,
        pointerCount: pointerCount,
      });

      if (gestureType === 'pan') {
        emit(el, 'st:pan:start', { x: e.clientX, y: e.clientY });
      } else if (gestureType === 'pinch') {
        emit(el, 'st:pinch:start', {
          scale: 1,
          originX: (pointers[0].clientX + pointers[1].clientX) / 2,
          originY: (pointers[0].clientY + pointers[1].clientY) / 2,
        });
      } else if (gestureType === 'rotate') {
        emit(el, 'st:rotate:start', {
          angle: state.startAngle,
        });
      }
    }

    function onPointerMove(e) {
      if (!state.active) return;

      // Update pointer entry
      if (state.pointers[e.pointerId]) {
        state.pointers[e.pointerId] = {
          clientX: e.clientX,
          clientY: e.clientY,
          time: Date.now(),
        };
      }

      var pointers = getActivePointers();
      if (pointers.length < 1) return;

      var currentPoint = { x: e.clientX, y: e.clientY };

      // Calculate deltas
      var deltaX = currentPoint.x - state.lastPoint.x;
      var deltaY = currentPoint.y - state.lastPoint.y;
      var dt = Date.now() - state.startTime;

      state.totalDeltaX += deltaX;
      state.totalDeltaY += deltaY;

      if (dt > 0) {
        state.velocityX = state.totalDeltaX / dt;
        state.velocityY = state.totalDeltaY / dt;
      }

      // Build detail payload
      var detail = {
        type: gestureType,
        x: currentPoint.x,
        y: currentPoint.y,
        deltaX: deltaX,
        deltaY: deltaY,
        totalDeltaX: state.totalDeltaX,
        totalDeltaY: state.totalDeltaY,
        velocityX: state.velocityX,
        velocityY: state.velocityY,
        pointerCount: pointerCount,
        elapsed: dt,
      };

      // Pinch-specific data
      if (gestureType === 'pinch' || gestureType === 'pinch-rotate') {
        if (pointers.length === 2) {
          var currentDistance = distance(pointers[0], pointers[1]);
          detail.scale =
            state.startDistance > 0
              ? currentDistance / state.startDistance
              : 1;
          var mid = midpoint(pointers[0], pointers[1]);
          detail.originX = mid.x;
          detail.originY = mid.y;

          // Update CSS custom properties for pinch
          setCSSVar(el, '--st-pinch-scale', detail.scale.toFixed(4));
          setCSSVar(el, '--st-pinch-origin-x', detail.originX + 'px');
          setCSSVar(el, '--st-pinch-origin-y', detail.originY + 'px');
        }
      }

      // Rotate-specific data
      if (gestureType === 'rotate' || gestureType === 'pinch-rotate') {
        if (pointers.length === 2) {
          var currentAngle = angle(pointers[0], pointers[1]);
          detail.angle = currentAngle - state.startAngle;

          // Update CSS custom property for rotation
          setCSSVar(el, '--st-rotate-angle', detail.angle.toFixed(2) + 'deg');
        }
      }

      // Update CSS custom properties for pan position
      setCSSVar(el, '--st-pan-x', state.totalDeltaX.toFixed(2) + 'px');
      setCSSVar(el, '--st-pan-y', state.totalDeltaY.toFixed(2) + 'px');
      setCSSVar(el, '--st-pan-velocity-x', state.velocityX.toFixed(4));
      setCSSVar(el, '--st-pan-velocity-y', state.velocityY.toFixed(4));

      state.lastPoint = currentPoint;

      // Emit move events
      emit(el, 'st:gesture:move', detail);

      if (gestureType === 'pan') {
        emit(el, 'st:pan:move', detail);
      } else if (gestureType === 'pinch' || gestureType === 'pinch-rotate') {
        emit(el, 'st:pinch:move', detail);
      }
      if (gestureType === 'rotate' || gestureType === 'pinch-rotate') {
        emit(el, 'st:rotate:move', detail);
      }
    }

    function onPointerUp(e) {
      delete state.pointers[e.pointerId];
      pointerCount = Object.keys(state.pointers).length;

      // Continue if there are remaining pointers (e.g., pinch → pan)
      if (pointerCount >= 1 && state.active) {
        var remainingGestures = classifyGesture(pointerCount);
        if (remainingGestures) {
          gestureType = remainingGestures;

          // Recalculate start state for the new gesture type
          state.totalDeltaX = 0;
          state.totalDeltaY = 0;
          state.startTime = Date.now();

          var pointers = getActivePointers();
          if (pointers.length === 1) {
            state.lastPoint = { x: pointers[0].clientX, y: pointers[0].clientY };
            state.startPoint = state.lastPoint;
          }
          return;
        }
      }

      if (!state.active) return;

      // Calculate final velocity and determine swipe
      var dt = Date.now() - state.startTime;
      var finalVelocityX = dt > 0 ? state.totalDeltaX / dt : 0;
      var finalVelocityY = dt > 0 ? state.totalDeltaY / dt : 0;
      var finalDetail = {
        type: gestureType,
        totalDeltaX: state.totalDeltaX,
        totalDeltaY: state.totalDeltaY,
        velocityX: finalVelocityX,
        velocityY: finalVelocityY,
        elapsed: dt,
        pointerCount: pointerCount,
      };

      // Emit end event
      emit(el, 'st:gesture:end', finalDetail);

      if (gestureType === 'pan') {
        emit(el, 'st:pan:end', finalDetail);

        // Check for swipe
        if (state.enabledGestures.indexOf('swipe') !== -1) {
          var absVX = Math.abs(finalVelocityX);
          var absVY = Math.abs(finalVelocityY);

          if (absVX > SWIPE_VELOCITY_THRESHOLD || absVY > SWIPE_VELOCITY_THRESHOLD) {
            var swipeDetail = {
              velocityX: finalVelocityX,
              velocityY: finalVelocityY,
              speed: Math.sqrt(finalVelocityX * finalVelocityX + finalVelocityY * finalVelocityY),
            };

            if (absVX > absVY) {
              if (finalVelocityX > 0) {
                emit(el, 'st:swipe:right', swipeDetail);
              } else {
                emit(el, 'st:swipe:left', swipeDetail);
              }
            } else {
              if (finalVelocityY > 0) {
                emit(el, 'st:swipe:down', swipeDetail);
              } else {
                emit(el, 'st:swipe:up', swipeDetail);
              }
            }
          }
        }
      } else if (gestureType === 'pinch' || gestureType === 'pinch-rotate') {
        emit(el, 'st:pinch:end', finalDetail);
      }
      if (gestureType === 'rotate' || gestureType === 'pinch-rotate') {
        emit(el, 'st:rotate:end', finalDetail);
      }

      // Reset state
      state.active = false;
      gestureType = null;
      el.classList.remove('st-gesture-active');

      // Release pointer capture
      try { el.releasePointerCapture(e.pointerId); } catch (err) { /* no-op */ }
    }

    function onPointerCancel(e) {
      onPointerUp(e);
    }

    // --- Tilt tracking (pointer hover) ---
    function onPointerEnter(e) {
      if (state.enabledGestures.indexOf('tilt') === -1) return;
    }

    function onTiltMove(e) {
      if (state.enabledGestures.indexOf('tilt') === -1) return;

      var rect = el.getBoundingClientRect();
      var centerX = rect.left + rect.width / 2;
      var centerY = rect.top + rect.height / 2;

      // Calculate tilt angle based on pointer position relative to centre
      var relativeX = (e.clientX - centerX) / (rect.width / 2);
      var relativeY = (e.clientY - centerY) / (rect.height / 2);

      var maxTilt = parseFloat(
        getComputedStyle(el).getPropertyValue('--st-tilt-max') || '20'
      );

      var rotateY = relativeX * maxTilt;
      var rotateX = -relativeY * maxTilt;

      setCSSVar(el, '--st-tilt-rotate-x', rotateX.toFixed(2) + 'deg');
      setCSSVar(el, '--st-tilt-rotate-y', rotateY.toFixed(2) + 'deg');

      emit(el, 'st:tilt:move', {
        rotateX: rotateX,
        rotateY: rotateY,
        relativeX: relativeX,
        relativeY: relativeY,
      });
    }

    function onPointerLeave(e) {
      if (state.enabledGestures.indexOf('tilt') === -1) return;
      // Reset tilt
      setCSSVar(el, '--st-tilt-rotate-x', '0deg');
      setCSSVar(el, '--st-tilt-rotate-y', '0deg');
    }

    // --- Bind / Unbind ---
    function bind() {
      el.addEventListener('pointerdown', onPointerDown);
      el.addEventListener('pointermove', onPointerMove);
      el.addEventListener('pointerup', onPointerUp);
      el.addEventListener('pointercancel', onPointerCancel);
      el.addEventListener('pointerleave', onPointerLeave);

      if (state.enabledGestures.indexOf('tilt') !== -1) {
        el.addEventListener('pointerenter', onPointerEnter);
        el.addEventListener('pointermove', onTiltMove);
        el.addEventListener('pointerleave', onPointerLeave);
      }
    }

    function unbind() {
      el.removeEventListener('pointerdown', onPointerDown);
      el.removeEventListener('pointermove', onPointerMove);
      el.removeEventListener('pointerup', onPointerUp);
      el.removeEventListener('pointercancel', onPointerCancel);
      el.removeEventListener('pointerleave', onPointerLeave);
      el.removeEventListener('pointerenter', onPointerEnter);
      el.removeEventListener('pointermove', onTiltMove);
      el.removeEventListener('pointerleave', onPointerLeave);
    }

    return {
      bind: bind,
      unbind: unbind,
    };
  }

  // --- Global Initialisation ---

  var trackers = new Map();

  /**
   * Initialise gesture tracking on all [data-st-gesture] elements.
   */
  function initGestures() {
    var elements = document.querySelectorAll('[data-st-gesture]');

    elements.forEach(function (el) {
      if (trackers.has(el)) return;

      var tracker = createGestureTracker(el);
      tracker.bind();
      trackers.set(el, tracker);
    });
  }

  /**
   * Teardown an element.
   */
  function destroyGesture(el) {
    var tracker = trackers.get(el);
    if (tracker) {
      tracker.unbind();
      trackers.delete(el);
    }
  }

  /**
   * Re-initialise after DOM changes.
   */
  function refresh() {
    // Unbind removed elements
    trackers.forEach(function (tracker, el) {
      if (!document.contains(el)) {
        tracker.unbind();
        trackers.delete(el);
      }
    });

    // Bind new elements
    initGestures();
  }

  // --- Bootstrap ---
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initGestures);
  } else {
    initGestures();
  }

  // Watch for dynamically added elements
  var domObserver = new MutationObserver(function () {
    refresh();
  });

  domObserver.observe(document.body, {
    childList: true,
    subtree: true,
  });

  /**
   * Public API.
   */
  window.StydeGesture = {
    init: initGestures,
    destroy: destroyGesture,
    refresh: refresh,
    SWIPE_VELOCITY_THRESHOLD: SWIPE_VELOCITY_THRESHOLD,
  };
})();
```

### 6.3 Event Bus Reference

| Event | Source | Detail |
|-------|--------|--------|
| `st-vt:start` | ViewTransitions | `{ url, type, direction }` |
| `st-vt:complete` | ViewTransitions | `{ url, type }` |
| `st-vt:error` | ViewTransitions | `{ url, error }` |
| `st-vt:content-updated` | ViewTransitions | `{ url }` |
| `st:gesture:start` | Gesture Engine | `{ type, x, y, pointerCount }` |
| `st:gesture:move` | Gesture Engine | `{ type, deltaX, deltaY, velocityX, velocityY, ... }` |

## 7. Performance & Browser Compatibility

### 7.1 Performance Guidelines

| Rule | Rationale |
|------|-----------|
| **Use `will-change` sparingly** | Apply only during active gesture/transition; remove after. The gesture engine adds `.st-gesture-active` for this. |
| **Prefer `transform` + `opacity`** | These are the only properties that run on the compositor thread. All animations in this system use exclusively these properties. |
| **Passive scroll listeners** | The ScrollTimeline JS shim uses `{ passive: true }` on all scroll listeners. No `preventDefault()` calls. |
| **`requestAnimationFrame` batching** | FLIP engine uses double `rAF` to ensure layout has settled before playing. |
| **`IntersectionObserver` for view detection** | No scroll event handlers for view-progress tracking. |
| **Throttled gesture updates** | Pointer events fire at native device rate (~60–120Hz). CSS custom property updates are inherently throttled by the browser's style recalc cycle. |
| **Avoid layout thrashing** | FLIP engine reads all bounding rects in one phase (`capture`) then writes transforms in another (`animate`). |
| **Clean up observers** | MutationObservers disconnect when elements are removed. |
| **Minimal GC pressure** | Object pooling not used (simplicity over micro-optimisation), but no allocations inside animation frames. |

### 7.3 Graceful Degradation Summary

- **ScrollTimeline**: Falls back to JS-driven custom property updates using IntersectionObserver + scroll listeners.
- **View Transitions**: Falls back to CSS overlay fades with content swap via `fetch` + `DOMParser`.
- **FLIP**: Pure JS, works everywhere with identical behaviour.
- **Gesture Engine**: Pure Pointer Events API, works everywhere. Touch gestures degrade naturally on non-touch devices (keyboard/mouse interactions are not affected).
