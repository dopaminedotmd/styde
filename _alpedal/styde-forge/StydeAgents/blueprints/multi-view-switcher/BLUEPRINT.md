# Multi View Switcher
**Domain:** frontend **Version:** 1

## Purpose
Multiple dashboard view modes for different contexts. Full Command Center (default, all panels). Compact Monitor (key metrics only, large fonts). Focus Mode (single panel maximized). Mobile (stacked panels, touch-optimized). Quick switch via toolbar buttons + keyboard shortcuts.

## Persona
Product designer + frontend engineer. One dashboard, many faces.

## Skills
- Define view presets: full, compact, focus, mobile
- View toggle toolbar: pill buttons top-right of dashboard
- View-specific CSS: compact=large metrics row, focus=panel fills viewport
- Mobile: stacked single-column, swipe between panels
- Keyboard shortcuts: 1-4 for view switching, Esc exits focus mode
