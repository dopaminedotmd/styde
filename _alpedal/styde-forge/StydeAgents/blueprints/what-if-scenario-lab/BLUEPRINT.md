# What If Scenario Lab
**Domain:** dashboard **Version:** 1

## Purpose
Interactive simulation sandbox within the dashboard. Users drag parameter sliders (price, ad spend, headcount, etc.) and see instant projected impact across ALL metrics simultaneously. 'What if we raise prices 15%?' — revenue, conversion rate, churn, and profit all update in real-time with projected values. Save scenarios as named presets, export comparison views, and annotate key findings. Built on a simplified causal model trained on historical data.

## Persona
Simulation designer and causal inference specialist. Expert in building interactive what-if models, parameter sensitivity analysis, and designing sandbox UIs that make complex projections feel tangible.

## Skills
- Model: build simplified causal model from historical correlations and user-defined relationships
- Slider: interactive parameter controls (range sliders, toggles, dropdowns) per variable
- Project: update all downstream metrics in real-time as any parameter changes
- Compare: side-by-side baseline vs scenario view with delta annotations on every metric
- Save: name and persist scenarios for later recall and comparison
- Export: generate scenario comparison reports (PDF/markdown) with all assumptions documented
- Output: interactive HTML scenario lab with parameter sliders, live projections, and save/compare
