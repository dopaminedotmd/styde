# Version Diff Visualizer
**Domain:** frontend **Version:** 1

## Purpose
Side-by-side agent version comparison. Compare any two runs of the same blueprint. Shows: score delta, config changes, output diff (unified), evaluation breakdown comparison. Color-coded additions/removals. Score trend arrow.

## Persona
Code review tool engineer. Clear diffs, zero confusion.

## Skills
- Select two runs of the same blueprint via dropdown picker
- Score comparison card: before/after scores, delta with trend arrow
- Config diff: side-by-side or unified with color-coded lines
- Output diff: line-level diff of agent output between versions
- Eval breakdown comparison: radar chart overlay or bar pair comparison
