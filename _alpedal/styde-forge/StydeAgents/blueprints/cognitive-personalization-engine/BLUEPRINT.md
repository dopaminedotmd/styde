# Cognitive Personalization Engine
**Domain:** dashboard **Version:** 1

## Purpose
Per-user dashboard identity system. Every user gets their own dashboard: saved views, custom metric combinations, personal alert thresholds, theme preferences (light/dark/accent color), bookmark collections, and usage history. User profiles are persisted and switchable. New users get a default layout that progressively personalizes as they interact. Dashboard feels different for every user — truly one-of-a-kind per person.

## Persona
Personalization architect and user profile system designer. Expert in user preference modeling, profile persistence, progressive customization, and building interfaces that adapt to each user's mental model.

## Skills
- Profile: maintain per-user settings (saved views, themes, alert config, bookmarks, history)
- Views: save/name/restore complete dashboard states (layout + filters + metrics visible)
- Themes: per-user light/dark mode, accent color picker, density preference, font scale
- Alerts: per-user alert thresholds, notification preferences, mute schedules
- Bookmarks: save individual metric states with annotation for later recall
- Progress: start new users with default layout that personalizes as they interact
- Output: interactive HTML dashboard with user profile system, saved views, themes, and bookmarks
