# Config Reference

**Styde Forge v3.0 — Phase 0**
**Section:** 00_Overview

---

## 1. Purpose

Complete reference for all configuration parameters in the forge.
Every configurable value, its default, valid range, and where it's stored.

---

## 2. Global State (`state.yaml`)

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `forge_version` | string | `"3.0.0"` | Forge version |
| `forge_codename` | string | `"The Crucible"` | Release codename |
| `created` | ISO 8601 | auto | Creation timestamp |
| `last_checkpoint` | ISO 8601 | auto | Last checkpoint timestamp |
| `hardware_profile` | string | `"pontus-main"` | Active hardware profile |
| `loop_iterations` | int | `0` | Total loop iterations completed |
| `total_agents_spawned` | int | `0` | Total agents ever spawned |
| `total_evaluations` | int | `0` | Total evaluations completed |

---

## 3. Blueprint Config (`blueprints/<name>/config.yaml`)

### blueprint section

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `blueprint.name` | string | required | Blueprint name |
| `blueprint.version` | int | `1` | Blueprint version |
| `blueprint.description` | string | `""` | Human-readable description |
| `blueprint.domain` | string | optional | Domain (coding, research, etc.) |

### hardware_profiles section

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `hardware_profiles.<profile>.model` | string | `"deepseek-v4-flash"` | Agent spawn model |
| `hardware_profiles.<profile>.provider` | string | `"deepseek"` | Provider name |
| `hardware_profiles.<profile>.eval_model` | string | `"deepseek-v4-pro"` | Judge/Teacher model |
| `hardware_profiles.<profile>.max_tokens` | int | `8192` | Max output tokens |
| `hardware_profiles.<profile>.temperature` | float | `0.3` | Model temperature |

### agent section

| Key | Type | Default | Range | Description |
|-----|------|---------|-------|-------------|
| `agent.max_iterations` | int | `10` | 1-50 | Max agent turns |
| `agent.timeout_seconds` | int | `300` | 30-600 | Agent timeout |
| `agent.retry_on_failure` | bool | `true` | — | Auto-retry on failure |
| `agent.toolsets` | list | `["terminal","file","web"]` | — | Tools available to agent |

### eval section

| Key | Type | Default | Range | Description |
|-----|------|---------|-------|-------------|
| `eval.benchmarks` | list | `[]` | — | Benchmarks to run |
| `eval.judge_model` | string | `"deepseek-v4-pro"` | Model for LLM-as-Judge (always Pro) |
| `eval.min_pass_score` | float | `0.70` | 0.0-1.0 | Minimum score to pass |
| `eval.max_retries` | int | `3` | 1-10 | Max retries on score 70-79 |
