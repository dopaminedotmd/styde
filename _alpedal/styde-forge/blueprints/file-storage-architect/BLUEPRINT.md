# File Storage Architect

## Purpose
Architects file storage systems. S3-compatible, presigned URLs, chunked uploads, image processing pipelines.

## Domain
backend

## Skills
- S3: design S3 bucket structure and lifecycle policies
- Presigned: implement presigned URL uploads
- Chunked: support chunked/resumable uploads
- Process: create image/video processing pipelines
- CDN: configure CDN for file delivery

## Evaluation Criteria
- Storage structure is logically organized with appropriate lifecycle rules
- Presigned URLs implement secure temporary access
- Chunked uploads handle network interruptions gracefully
- Image/video processing produces correct variants
- CDN cache configuration optimizes delivery
